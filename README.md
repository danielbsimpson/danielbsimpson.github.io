## About Me

I'm a Data Science Manager at The TJX Companies based in Framingham, Massachusetts, leading a team to deliver data science and AI solutions across the US Consumer business while supporting European operations. I hold an MSc in Data Science (Distinction) from Birkbeck, University of London, and a BSc in Mathematics from West Virginia University.

My work spans predictive modelling, machine learning, NLP, and AI tooling — with a focus on enabling teams to adopt modern data platforms and ways of working.

## About This Site

This repository hosts my personal portfolio website, showcasing both professional work projects and personal data science projects published on GitHub. It includes an interactive resume, an MSc project write-up, and several embedded web applications built entirely client-side.

## How It's Built

- **Base theme:** [Forty by HTML5 UP](https://html5up.net) (free under CCA 3.0)
- **Frontend:** Vanilla HTML, CSS, and JavaScript — no frameworks, no backend
- **Styling:** Custom dark space-themed design layered over the base theme using CSS custom properties, with [Orbitron](https://fonts.google.com/specimen/Orbitron) and [Exo 2](https://fonts.google.com/specimen/Exo+2) typefaces
- **Data visualisation:** [Chart.js](https://www.chartjs.org) and [D3.js](https://d3js.org) for interactive charts
- **Hosting:** GitHub Pages
