Facial Recognition & Mask Detection

  • View on GitHub
  • ← Back to Projects

OpenCV face detection feeding a Keras CNN classifier — built during the height of the pandemic.

Python OpenCV Keras TensorFlow CNN ImageNet Transfer Learning

What It Is

A two-stage computer vision pipeline: OpenCV first detects and isolates individual faces in an image using a Haar cascade classifier, then feeds each cropped face into a Keras convolutional neural network built on ImageNet weights to classify whether the person is wearing a face mask or not.

The pipeline can process both still images and live video streams, drawing bounding boxes and labelling each face with a confidence score in real time.

Why I Built It

This project was built in 2020 during the early stages of the Covid-19 pandemic, when mask compliance was a genuine public health concern and computer vision solutions for automated detection were a real area of interest.

More broadly, computer vision has always been one of my deepest interests in AI. The idea that a model can learn to see — to extract meaningful structure from raw pixels — connects directly to the questions I've thought about since studying neurobiology: how does the visual cortex build representations of the world from photoreceptor signals? Convolutional neural networks are a direct computational analogy to that biological architecture, and I find that genuinely fascinating.

This project was my first real hands-on experience combining two vision techniques — classical detection with OpenCV and learned classification with a CNN — into a single usable pipeline.

How It Works

  • Detection

    OpenCV Haar Cascades — A pre-trained Haar cascade classifier detects face regions in the input image, returning bounding box coordinates for each detected face. This is fast and reliable for frontal faces.

  • Crop

    Face Extraction — Each detected face region is cropped, resized to the model's expected input dimensions, and normalised before being passed to the CNN.

  • Classify

    Keras CNN on ImageNet — A CNN trained with transfer learning from ImageNet weights classifies each face as masked or unmasked, returning a probability score for each class.

  • Output

    Annotated Visualisation — The original image is returned with bounding boxes colour-coded by classification result and labelled with the model's confidence score.

Companion Project

This project has a companion: CNN & Transfer Learning Comparison, which benchmarks two separate CNN architectures for the mask detection task — one trained from scratch and one using InceptionResNetV2 transfer learning — to quantify the efficiency gains from pre-trained weights on a constrained dataset.
