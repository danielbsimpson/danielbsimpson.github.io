Family Tree

  • Open Family Tree
  • ← Back to Projects

An interactive, client-side family tree visualisation — no database, no backend, just data and a browser.

JavaScript D3.js HTML CSS JSON

What It Is

The Family Tree is an interactive ancestry visualisation built as a standalone web page. It renders the family's relationships as an expandable, navigable node graph — allowing you to trace lineages, explore branches, and view biographical details for individual family members, all from a single HTML file with JSON data.

The tree is rendered entirely client-side using D3.js, with no backend server, database, or external API calls. The data is stored in a structured JSON file that can be updated directly.

Features

  • Node Graph

    Collapsible D3.js tree layout — expand and collapse branches to navigate across generations without visual clutter.

  • Profiles

    Click any node to view biographical details: name, birth year, relationships, and any additional notes stored in the JSON data.

  • Zoom & Pan

    Full zoom and pan support for navigating large trees without losing context of the broader structure.

  • Data-Driven

    The entire tree is driven by a single JSON file — adding new family members or relationships requires only a data update, not code changes.

Why I Built It

My family has traced its lineage back several generations across multiple countries, and I wanted a way to visualise that history that was more interactive and navigable than a printed chart. Commercial genealogy tools either require subscriptions or lock data into proprietary formats — building my own meant I owned the data and the presentation entirely.

It also gave me a good excuse to work with D3's tree layout, which is a genuinely elegant solution to a complex hierarchical visualisation problem.
