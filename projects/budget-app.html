Personal Budget Dashboard

  • Open Dashboard
  • ← Back to Projects

A fully client-side personal finance dashboard — track spending, savings, and income with no server and no data leaving your browser.

JavaScript Chart.js HTML CSS JSON

What It Is

The Personal Budget Dashboard is an interactive finance tracker built entirely in the browser. It visualises income, expenses, and savings across configurable categories using dynamic Chart.js charts — with no backend server, no database, and no data ever leaving the client.

Transactions are loaded from a JSON data file and the dashboard updates all charts and summaries in real time. The sidebar breaks down spending by category, and the main view shows monthly trends across the full period.

Features

  • Overview

    At-a-glance summary of total income, total expenses, and net savings for the selected period — updated dynamically as filters change.

  • Category Breakdown

    Configurable spending categories with doughnut and bar chart visualisations — see exactly where money is going each month.

  • Trend Analysis

    Line charts showing income and expense trends over time, making it easy to spot months where spending spikes or savings improve.

  • Fully Client-Side

    All data stays in the browser. The app runs from static files with no server, making it easy to self-host or run locally from a single folder.

Why I Built It

Most personal finance apps require a bank connection, a subscription, or the trust that your financial data is being handled securely by a third party. I wanted something simpler: a tool I could run locally, load my own exported transaction data into, and use entirely offline. Privacy by design, not as a feature.

It also gave me an opportunity to work with Chart.js in depth — exploring how to build responsive, animated financial charts that update cleanly when data or filter state changes.
